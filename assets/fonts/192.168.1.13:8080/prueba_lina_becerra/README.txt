Overview

En este ejercicio el candidato deberá construir el feed de OWAK Music a partir de un llamado AJAX, basando el diseño en la UI entregada.

Guía

- NO se deben usar frameworks o librerías de JavaScript como jQuery, Angular, React.
- Se aceptan pre-compiladores de CSS como LESS y SASS, pero no librerías como Bootstrap o Foundation.
- Debe ser responsive
- La barra de compartir NO debe funcionar

Nos gustaría

- Documentación
- Propuesta de animaciones

Archivos

- Mockup: PDF describiendo como debería verse el feed de OWAK Music
- feed.json: Contiene la información que deberá ser mostrada en el diseño

Especificaciones de Diseño
———

Tipografía

- Títulos: Brandon Grotesque Black 31.7pts
- Artistas: Calibri 18pts 

Color

- Orange: #f05c28
- Yellow: #fcb614
- Light Gray: #cccccc
- Translucent Black: rgba(0,0,0,0.35)
- Aqua: #00d2b0

Interacciones

- Al hacer scroll la barra de compartir se debe mantener on top
- On hover el botón de SHARE debe presentar algún feedback
- On Hover, debe aparecer el recuadro Aqua al rededor del item
- Al dar click en la “x” que aparece On Hover se debe eliminar el elemento del JSON
- Al dar click en “Play >” debe aparecer y mantenerse el recuadro degradado sobre el item
